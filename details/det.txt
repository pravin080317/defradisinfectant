<section class="govuk-width-container">
  <h1 class="govuk-heading-xl">Accessibility statement for the local authority portal</h1>

  <p class="govuk-body">
    The Valuation Office Agency (VOA) is committed to making its websites accessible, in accordance with the Public Sector Bodies (Websites and Mobile Applications) (No. 2) Accessibility Regulations 2018.
  </p>

  <p class="govuk-body">
    This accessibility statement applies to 
    <a href="https://voabsportaluat.powerappsportals.com" class="govuk-link">https://voabsportaluat.powerappsportals.com</a>.
  </p>

  <h2 class="govuk-heading-l">Using this service</h2>

  <p class="govuk-body">
    This service allows billing authorities and local authorities to exchange property information with the VOA.
  </p>

  <p class="govuk-body">
    We want as many people as possible to be able to use this service. For example, that means you should be able to:
  </p>

  <ul class="govuk-list govuk-list--bullet">
    <li>change colours, contrast levels and fonts using browser or device settings</li>
    <li>zoom in up to 400% without the text spilling off the screen</li>
    <li>navigate most of the website using a keyboard, including tabbing through links and form fields</li>
    <li>interact with all page elements using assistive technologies such as screen readers (JAWS, NVDA, VoiceOver)</li>
    <li>submit forms and access buttons without using a mouse</li>
    <li>use voice input and speech recognition software for navigation and data entry</li>
  </ul>

  <p class="govuk-body">
    We’ve also made the website text as simple as possible to understand.
  </p>

  <p class="govuk-body">
    <a href="https://mcmw.abilitynet.org.uk/" class="govuk-link" rel="noreferrer noopener" target="_blank">
      AbilityNet has advice
    </a> on making your device easier to use if you have a disability.
  </p>

  <h2 class="govuk-heading-l">Accessibility standards and compatibility</h2>

  <p class="govuk-body">
    This site is designed to meet the <strong>WCAG 2.2 AA standard</strong> and works with the following assistive technologies:
  </p>

  <ul class="govuk-list govuk-list--bullet">
    <li>Screen readers: JAWS, NVDA, and VoiceOver</li>
    <li>Screen magnifiers and browser zoom tools</li>
    <li>Keyboard-only navigation</li>
    <li>Voice control tools such as Dragon NaturallySpeaking</li>
    <li>Browser accessibility features (e.g., tab order, skip links, ARIA labels)</li>
  </ul>

  <h2 class="govuk-heading-l">Compliance status</h2>

  <p class="govuk-body">
    This website is fully compliant with the Web Content Accessibility Guidelines (WCAG) version 2.2 AA standard.
  </p>

  <h2 class="govuk-heading-l">Preparation of this accessibility statement</h2>

  <p class="govuk-body">
    This statement was prepared on 30 June 2025.
  </p>

  <p class="govuk-body">
    The Digital Inclusion and Accessibility Standards team conducted a WCAG 2.2 AA accessibility audit retest of the VOA Local Authority (LA) portal during the week commencing Monday 23 June 2025.
  </p>

  <p class="govuk-body">
    All previously reported WCAG 2.2 issues have been fixed and verified as fixed during this retest. Two new WCAG issues were also identified but they were fixed and verified as fixed during this retest.
  </p>

  <p class="govuk-body">
    Under the 
    <a href="https://www.legislation.gov.uk/uksi/2018/952/contents/made" class="govuk-link" target="_blank">
      Regulations to make public sector websites and mobile applications accessible
    </a> 
    this service can publish a fully compliant WCAG 2.2 AA accessibility statement.
  </p>

  <p class="govuk-body">
    The statement was last reviewed on 30 June 2025.
  </p>
