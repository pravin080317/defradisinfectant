Frontend: Microsoft Power Apps Portal (Power Pages), enhanced with HTML/CSS/JavaScript following Gov.UK Design System standards.

Backend: Microsoft Dataverse with C# plugins and Azure Functions to handle business logic and file processing.

Integration: Azure APIM, Azure Blob Storage, SDES, and the Integration Framework for ECS/HMRC interactions.

Deployment: Azure DevOps pipelines using Power Platform Build Tools and ARM templates.

Third Parties: Microsoft SaaS platforms (Power Platform, Azure), virus scanning via SDES (Defra service), Exchange Online for notifications.
