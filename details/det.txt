### **ðŸ§  How the Retrieval Will Work**

#### 1. **In the Power Pages HTML / Template**

Add a `data-i18n-id` attribute to each content block:

```html
<h1 id="header-title" data-i18n-id="header-title">Local Authority Portal</h1>
<p id="header-subtitle" data-i18n-id="header-subtitle">Welcome to the online service</p>
```

#### 2. **On Page Load or Toggle**

When the user selects **Cymraeg**, a lightweight JS retrieves Welsh content:

```js
const pageUrl = window.location.pathname;
fetch(`/api/data/v9.2/welshtranslations?$filter=pageurl eq '${pageUrl}'`)
  .then(res => res.json())
  .then(data => {
    data.value.forEach(item => {
      const element = document.querySelector(`[data-i18n-id="${item.contentkey}"]`);
      if (element && item.welshcontent) {
        element.innerHTML = item.welshcontent;
      }
    });
    document.documentElement.lang = "cy"; // update <html lang>
  });
```
