### ‚úÖ What Power Apps Map Control **CAN DO** (as of 2025)

1. **Display Multiple Pins**:

   * You can show 1,000+ pins based on latitude/longitude from a data source (Dataverse, Collection, SharePoint, etc.).

2. **Custom Pin Colours**:

   * You can bind each pin‚Äôs colour to a field like `Band` (e.g., Red for A, Blue for B, etc.).
   * Use the `Color` property within your record‚Äôs JSON.

3. **Tooltip & Label on Click**:

   * When a user clicks a pin, it can show info like UARN, address, price.

4. **Zoom, Pan, Cluster**:

   * Supports pan/zoom.
   * Can auto-cluster if pin density is high.

---

### ‚ùå What It **CANNOT DO** Directly

| Limitation                      | Explanation                                                                                                                                                                                         |
| ------------------------------- | --------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| **Layer multiple data sources** | You can only bind **one data source at a time** to the Map Control. <br>So if you want to show properties from both "To Work" and "Paused", they must be merged into a **single collection first**. |
| **Built-in Legend**             | The Map Control does **not support a visible legend** (e.g., Red = A1, Blue = B2). You must **build your own legend using a `Gallery` control**.                                                    |
| **Conditional Icons or Shapes** | Only one marker shape/style is supported. You can‚Äôt change shape (e.g., circle vs triangle) ‚Äî just colour.                                                                                          |
| **Click Pin ‚Üí Filter Table**    | Requires extra logic: you must set a variable on pin click to filter the related `Gallery`.                                                                                                         |
| **No heatmaps or polygons**     | For advanced visuals (e.g., boundary overlays, zones), you need Power BI or PCF.                                                                                                                    |
